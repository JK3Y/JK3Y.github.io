how i do what i do

I like to use the LAMP stack, and I dabble in the MEAN stack.

Sublime Text is my preferred text editor. Recently I've been using WebStorm for all my web development, and PhpStorm for all my web + php work.

I use various CLI applications to make life a little bit easier for me:

  • Oh My Zsh to create the best terminal in the world
  • Homebrew is my one-stop shop for package updates
  • npm for all my node.js package needs
  • Bower to get all my front-end dependencies
  • Git for use with GitHub

XCode 9 for Swift development

Visual Studio 2012 for C# and ASP.NET development.

Netbeans for Java development.
