XMRchant is a Laravel package that helps facilitate easier Monero payment integration for developers. Includes migration tables, a cron job to manage incoming transactions, clean easy-to-read account and payment data, and a facade for easy Laravel 5.5+ integration.

Chatr is a disposable chat application created using Node, Express, and Socket.io on the backend and jQuery, Bootstrap, and SCSS on the frontend. Chatrooms are created dynamically based on the URL. Hosted on Heroku.

BandcampDownloader is a basic web app used to download all the free tracks from an artist's Bandcamp page. User is able to change ID3 tags and even add album art. Exports the files in a simple zip file. Uses Bootstrap, file-saver, and jszip.
